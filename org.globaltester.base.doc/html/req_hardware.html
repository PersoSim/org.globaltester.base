Hardware

To run GlobalTester your system should have at least:

  • RAM: min. 2 GB
  • Hard disc: min. 400 MB
  • Processor operating frequency: 2 x 1.5 GHz dual core or more recommended

To test inspection systems you will need a hardware simulator. The following hardware simulators have been tested for their compatibility with GlobalTester:
